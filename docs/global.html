CookieBanner() → {JSX.Element|null}

CookieBanner component shows a banner at the bottom of the screen prompting the user to accept cookies if they haven't already given consent. Behavior: - On mount, checks localStorage for 'cookieConsent' key. - If consent not found, shows the banner. - When user clicks "Accept", stores consent in localStorage and hides the banner.
